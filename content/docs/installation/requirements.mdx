# Prerequisites

To set up Vercord, you will need the following:

1. A Vercel account with project(s).
2. A Discord server with the "Manage Webhooks" permission.
3. Your own Vercord instance:
   [![Deploy with Vercel](https://chmgukzv37.ufs.sh/f/m5aDeKXl439ZvyvPS1sp0jTNwxGyctDAnqWIrBf9kmC5zQ7K)](https://vercel.com/new/clone?repository-url=https://github.com/Vercord/vercord)
